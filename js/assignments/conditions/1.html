      Opdracht 6:

      We hebben nog een comparison operator niet gehad, namelijk ! comparison. Wat doet een !
      comparison ? Maak een voorbeeld van een ! comparison

(!) oftwel de logical NOT (!) operator neemt truth als falsy aan en vice versa. Het wordt altijd gebruikt in combinatie met een boolean (logical) values. Wanneer het wordt gebruikt met non-Boolean values, geeft het false terug als zijn enkele operand kan worden geconverteerd naar true; anders geeft true terug.

Returns false if its single operand can be converted to true; otherwise, returns true. If a value can be converted to true, the value is so-called truthy. If a value can be converted to false, the value is so-called falsy.

Voorbeelden van expressions dat geconverteerd kan worden naar false

  • null;
  • NaN;
  • 0;
  • empty string (" " of '' of ``);
  • undefined

Code

      n1 = !true               // !t returns false
      n2 = !false              // !f returns true
      n3 = !''                 // !f returns true
      n4 = !'Cat'
      console.log(n1)
      false
      console.log(n2)
      true
      console.log(n3)
      true
      console.log(n4)
      false

      Opdracht 8:
      
      Wat is een strict equality comparison ?

strict equality(===) controleert of de twee operanden gelijk zijn, en retourneert een Boolean als resultaat. In tegenstelling tot de equality operator beschouwt de strict equality operator operanden van verschillende typen altijd als verschillend.

De grootste verschil tussen === en == is dat wanneer de operanden van verschillende types zijn, de == een poging zal doen om hen om te zetten voordat er een vergelijking plaats vind.
