Vier voorbeelden van attributen die je een hyperlink kan meegeven

download

De download attribute specificeerd dat het doelwit van deze link gedownload zal worden als de gebruiker de hyperlink aanklinkt

    TG logo downloaden
target

De target attribute specificeerd waar gelinkte document wordt geopend, door bijv. _blank in te vullen open het doelwit in een nieuwe venster of tabblad

    Visit W3Schools
ping

De ping attribute specificeerd een lijst van URLs die genotificeerd worden als de gebruiker de hyperlink wilt tracen. Wanneer de gebruiker klikt op de hyperlink, zal de ping attribute een korte HTTP POST request sturen naar de gespecificeerde URL Deze attribute is handig voor monitoring/tracking

    w3schools
media

De media attribute geeft aan welke media of device de gelinkte document voor is geoptimaliseerd
