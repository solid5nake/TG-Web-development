HTML Forms

HTML formulieren zijn de belangrijkste schakel tussen een gebruiker en een website of een applicatie. Zij laten gebruikers toe gegevens naar websites te sturen. Meestal zal dat naar de server zijn maar een webpagina kan de gegevens ook zelf interpreteren. Een HTML form bestaat uit een aantal elementen. Bijvoorbeeld textfields (bestaande uit één of meerdere regels), choices of buttons.

Wanneer zou je een dropdownlist, radiobutton en checkbox gebruiken?

  • select element (dropdownlist), zou je bijvoorbeeld voor het selecteren een merk in een webshop of het kiezen van en kleur
  • input type radio (radiobutton), zou je bijvoorbeeld voor 'gender' kunnen gebruiken of bij een gesloten vraag (ja, nee)
  • input type checkbox (checkbox), zijn goed te gebruiken om een true/false(boolean) waarde terug te krijgen of wanneer je meerdere waardes van kunt kiezen uit een lijst
    • bijvoorbeeld als je kunt kiezen uit verschillende programmeertalen die men beheerst
