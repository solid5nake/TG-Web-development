Cascade

CSS is de afkorting voor Cascading Style Sheets, maar wat houd cascading in?
Op de meest simpele niveau wordt de CSS rules toegepast op volgorde; wanneer twee regels dezelfde specificaties hebben
Geldt de specificatie van de laatste regel.

In de onderstaande voorbeeld, hebben we twee css rules die toegepast kunnen worden op een h1
De header wordt uiteindelijk blauw. De regels hebben een identieke selector en hebben ook
dezelfde specifactie, de laatst genoemde wint van de twee.

Inheritence

In CSS, inheritance gebeurd wanneer er geen value is gespecificeerd op een
property van een element.

CSS properties kunnen gecategoriseerd worden in twee types:
  • inherited properties, by default wordt de waarde overgenomen van de parent
  • non-inherited properties, by default wordt de initieele waarde ingesteld
