Kim Sing's Array challenge

    Voor de geïnteresseerden onder ons, de opgave van Kim gaat als volgt;
    [1,5,9,7,6]  [8,4,2,10,11]
    Maak een functie aan die deze twee arrays meekrijgt als attributen.
    De output van deze function is één gesorteerde array.
    Je mag geen enkele array functies gebruiken, bijv het gebruik van  de array method .sort () is verboden.
    Enkel de array methode .length mag gebruikt worden.
  
    Er is gebruik gemaakt van de insertion sort algorithm. De insertion sortering maakt gebruik van een subarray.
    Met de algoritme wordt gecontroleerd of een getal naar links opgeschoven kan worden.
    Als het niet verder naar link moet dan gaat het over naar de volgende getal.

Code

    const numbers1 = [1, 5, 9, 7, 6];
    const numbers2 = [8, 4, 2, 10, 11];

    const combineArrays = (arr1, arr2) => {
      let array = [];

      for (let i = 0; i < arr1.length; i++) {
        array[array.length] = arr1[i];
      }
      for (let i = 0; i < arr2.length; i++) {
        array[array.length] = arr2[i];
      }
      for (let i = 1; i < array.length; i++) {
        let current = array[i];
        let j = i - 1;
        while ((j > -1) && (current < array[j])) {
          array[j + 1] = array[j];
          j--;
        }
        array[j + 1] = current;
      }
      return array;
    }
