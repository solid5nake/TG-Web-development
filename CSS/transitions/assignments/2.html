Transitions

Opdracht

Maak een simpele transition wat de kleur van een tekst aanpast.

Code

    HTML
    <p class="transition">
      Ga met de muis over mij heen om de kleur van de tekst te veranderen.
    </p>
  
    CSS
    .transition {
      transition-property: color;
      transition-duration: 3s;
    }

    .transition:hover {
      color: crimson;
    }

Resultaat

Dit is een voorbeeld van een transition waarbij de kleur van de tekst veranderd.
Ga met de muis over mij heen om de kleur van de tekst te veranderen.

:hover state op de paragraaf element toegepast hierdoor wordt de color veranderd van default naar de kleur crimson.
