Transitions

Opdracht 5

    Hieronder zie je wederom een voorbeeld van een transition. Je kunt op verschillende properties transitions aangeven.
    Zoek uit op welke properties de transitions zitten en maak het na. Je eigen creativiteit gebruiken is uiteraard
    toegestaan.
  
Ga met je muis over de joker kaart.
    De parent container heeft 3 children, waarvan alleen twee daarvan een transition hebben meegekregen. 
    On :hover state wordt de transition geactiveerd.

Code

    .joker {
      background: url(../images/joker-card.svg), rgb(190, 128, 255);
      background-size: 50%;
      transition: 5s transform ease 2s;
    }

    .crimson {
      transition: 1s background-color linear;
    }

    .haha-left {
      transform: translateX(0) rotate(0deg) translateY(46px);
    }

    .joker-haha {
      background-image: url(../images/joker-haha.jpg);
      background-size: cover;
    }

    .joker:hover {
      transform: translateX(230px) rotate(20deg) translateY(200px);
      transition: 3s transform ease 200ms;
    }

    .crimson>img {
      pointer-events: none;
      height: 39rem;
      opacity: 0;
      transition: 2s opacity ease-in;
    }

    .crimson:hover {
      background-color: rgba(220, 20, 60, 0.4);
      transition: 1s background-color linear;
    }

    .crimson:hover .haha-left,
    .crimson:hover .haha-right {
      opacity: 1;
      transition: 2s opacity linear;
    }
