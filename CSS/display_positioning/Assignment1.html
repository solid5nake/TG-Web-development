Opdracht 1

Welke position properties zijn er? Leg voor elke position property uit wat het doet.
Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

De position property stelt in hoe een element gepositioneerd wordt in een document.
Je hebt de volgende position properties; static, relative, absolute, fixed en sticky

static

div
.static
div

Het element is gepositioneerd volgens zijn normal flow van het document.
top, right, bottom, left en z-index properties hebben geen effect

relative

div
.relative
div
    CSS
    .relative {
      position: relative;
      bottom: 40px;
      left: 40px;
    }

Het element is verwijderd van de normal flow van het document, en dan offset
relatief tot zichzelf gebasseerd op top, right, bottom en left
De offset heeft geen invloed op de positioneering van andere elementen, netals
in een statische positioneering blijft de ruimte hetzelfde.

absolute

div
.absolute
div

Het element is verwijderd van de normal document flow en geen ruimte is gereserveerd voor het element.
Het wordt gepositioneerd relatief van de dichtsbijzijnde positioned ancestor. Netals bij de position property relative wordt het gepositioneerd aan de hand van de waarde van top, right, bottom en left

Sticky

div
.sticky
div

Een element met een position:sticky is gepositioneerd op basis van de gebruikers scroll positie.
Een sticky element schakelt tussen relative en fixed, afhankelijk van de scroll positie.
Het wordt relatief gepositioneerd todat een bepaalde offset is gehaald in de viewport - dan "sticked" het op zijn plaats zoals bij position:fixed.

fixed

div
.fixed
div
    HTML
    <div class="fixed-container">
      <div>div</div>
      <div class="fixed">.fixed</div>
      <div>div</div>
    </div>

In het deze voorbeeld heeft .fixed-container (ancestor) een property perspective hierdoor is .fixed niet gefixeerd rechtsonder van de viewport. 

    CSS
    .fixed-container {
      width: fit-content;
      display: flex;
      perspective: 10px;
    }

    .fixed {
      position: fixed;
      bottom: 40px;
      right: 40px;
    }

Een element met position: fixed; wordt verwijderd van de normal flow van het document. Het is relatief gepositioneerd van de viewport.
Dit houdt in dat het altijd op dezelfde plek begeeft zelfs als je aan het scrollen bent.

Behalve wanneer 1 van zijn ancestors een transform, perspective of filter property heeft met een andere waarde dan none
De positioneering wordt vastgesteld aan de waardes van top, right, bottom en left

Een positioned element is een element waarbij de berekende position value
relative, absolute, fixed of sticky is (met andere woorden alle values gelden behalve static. )
