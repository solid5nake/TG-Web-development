Opdracht 1

    Zie het onderstaande voorbeeld en maak het na met flex. Gebruik geen width, height of line-height.
1

3
4

Code

    <div class="container">
      <div class="div">1</div>
      <div class="div">2</div>
      <div class="div">3</div>
      <div class="div">4</div>
      <div class="div">5</div>
      <div class="div">6</div>
    </div>
  
    .container {
      display: flex;
      height: initial;
      justify-content: flex-start;
      margin: 50px 0 50px 0;
      background: rgb(251, 254, 183);
    }
    
    .div {
      display: flex;
      margin: 15px;
      flex-basis: 90px;
      block-size: 90px;
      justify-content: center;
      align-items: center;
      background: rgb(201, 251, 182);
      font-family: 'Times New Roman', Times, serif;
      font-size: 25px;
    }

Resultaat

1
2
3
4
5
6
    De parent heeft een display:flex property om het een flexbox van te maken. De child heeft een
    display:flex property om de tekst in het midden te centreren zonder gebruik te hoeven maken van
    line-height, de properties justify-content (horizontale as) en align-items (verticale as) worden ingesteld op center.
    De flex-basis property set de initial main size van een flex item (width).
    
    justify-content: flex-start; is de (default): items worden richting de start van de flex-direction ingepakt.
  
de block-size property is toegepast om de height in te stellen.

Opdracht 3

    Zie het onderstaande voorbeeld, een van de items is breder dan de rest. Maak het na met flex. Gebruik geen width,
    height of line-height.
1
2
3
4

Code

    <div class="container">
      <div class="div">1</div>
      <div class="flexgrow">2</div>
      <div class="div">3</div>
      <div class="div">4</div>
      <div class="div">5</div>
      <div class="div">6</div>
    </div>
  
    .flexgrow {
      flex-grow: 3;
    }

    div {
      display: flex;
      margin: 15px;
      flex-basis: 90px;
      block-size: 90px;
      justify-content: center;
      align-items: center;
      background: rgb(201, 251, 182);
      font-family: 'Times New Roman', Times, serif;
      font-size: 25px;
    }

    The flex-grow CSS property sets de flex grow factor van een flex item grootte. .flexgrow neemt de
    remaining space over 
    binnen de flexbox, gezien zijn grow factor 3 staat ten opzichte van de andere flex items die inngesteld staan op 1.
