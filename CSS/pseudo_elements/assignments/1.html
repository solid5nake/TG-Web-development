Pseudo elements

Opdracht 1

    Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen 
    woorden wat deze pseudo-elementen inhouden.
  
    Een CSS pseudo-element is een toegevoegde keyword aan een selector. 
    Het laat toe om een specifiek gedeelte te stylen van het geselecteerde element(en). 
    Alleen 1 pseudo-element kan gebruikt worden in een selector. Het moet achter de simple selectors aanwezig zijn binnen een declaratie.
  
    selector::pseudo-element {
      property: value;
    }

Index van standaard pseudo-elements

::first-line ::after ::before ::backdrop ::cue

::first-line

De ::first-line is een CSS pseudo-element dat gebruikt kan worden om de font van de eerste regel van een paragraaf first line te veranderen. 

    /* Geldt alleen voor de eerste regel van elke <p> */

    p::first-line {
      color: blue;
      text-transform: uppercase;
    }
  
*Geldt alleen voor de eerste regel van elke <p>

Note: Als regel worden dubbele colons :: gebruikt om onderscheidt te maken van pseudo-classes

::after

::after creeert een pseudo-element dat de last child is van een selected element Het wordt meestal gebruikt om cosmetische content toe te voegen aan een element met een content property. Per default is het inline. 

    / * Voeg een pijl toe achter elke link * /

    a::after {
      content: "➞";
    }

*Voeg een "➞" toe achter elke link

::before

::before creeert een pseudo-element dat dat de last child is van een selected element Het wordt meestal gebruikt om cosmetische content toe te voegen aan een element met een content property. Per default is het inline. 

    / * Voeg een hart toe achter elke link * /
    a::before {
      content: "♥";
    }

*Voeg een "♥" toe achter elke link

Note: pseudo-elementen gegenereerd met ::before en ::after worden contained door de elements formatting box om die reden wordt het niet toegepast door replaced elements zoals <img> en <br> elementen.

::backdrop

De ::backdrop pseudo-element is een box ter grootte van de viewport die onmiddellijk wordt weergegeven onder elk element dat in fullscreen mode wordt weergegeven, tegelijketijd zit het boven alle andre elementen. Dit omvat beide elementen die in volledig scherm zijn geplaatst met behulp van de Fullscreen API en <dialog> elementen. Wanneer meerdere elementen in de modus voor volledig scherm zijn geplaatst, wordt de achtergrond onmiddellijk onder het voorste element getekend, en bovenop de oudere elementen op volledig scherm. 

    dialog::backdrop {
      background: rgba(255,0,0,.25);
    }

*Backdrop wordt alleen weergegeven als het dialoogvenster wordt geopend met dialog.showModal ()

::cue

    Het :: cue CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element. Dit kan worden gebruikt
    om bijschriften en andere aanwijzingen in media te stylen met VTT-tracks. 
    Het primaire doel van WebVTT-bestanden is om tekstoverlays toe te voegen aan een <video>.

Note: Web Video Text Tracks Format is een formaat voor het weergeven van getimede teksttracks (zoals ondertitels of bijschriften) met behulp van het <track> element. 

    CSS
    ::cue {
      color: yellow;
      font-weight: bold;
    }
