De 3 verschillende selectors


  • type selector

    Een Type Selector (ook wel gerefereerd als een Element Type Selector) matched elements met de corresponderende element node name, zoals { p } , { span } ,{ div } tags. Type selectors worden in het algemeen gebruikt om in 1 keer veranderingen aan te brengen aan de style van een website


  • descendant selector

    Een Descendant Selector in CSS is een selector met een spatie tussen twee selectors zonder een combinator.


  • class selector

    Een class selector selecteerd alle elementen met een matchende class attribuut
