      Opdracht 2:
        Deze opdracht is een uitbreiding van opdracht 1.
        Programmeer de volgende logica in de checkNumber functie:
          ● Als het ingevoerde getal is groter dan 4, dan toon je op het label: “Het ingevoerde getal is groter dan 4”.

Code

      function checkNumber() {
        let input = document.getElementById('number').value;
        let text = document.getElementById('label');
        if (input == 3) {
          text.innerText = "Het ingevoerde getal is gelijk aan 3";
        }
        else
          if (input == 4) {
            text.innerText = "Het ingevoerde getal is gelijk aan 4";
          }
      }

else if statement is hierbij gebruikt om een tweede conitie mee te geven.
