CSS: Cascading Style Sheets

CSS is een stylesheet-language die wordt gebruikt om de presentatie van een document te beschrijven dat is geschreven in HTML of XML (inclusief XML-dialecten zoals SVG, MathML of XHTML).
CSS beschrijft hoe elementen op het scherm, op papier, in spraak of op andere media moeten worden weergegeven.
