Opdracht 7

Zie de onderstaande image met tekst. Kun jij dit namaken? Gebruik zelf een foto van een zonnige bestemming. Gebruik position, maar je mag geen z-index toepassen.

De teksten zijn ingedeeld in div.text en de img in div.box
div.text is absolute gepositioneerd van de ancestor div.box (position:relative) 

    CSS
    .box {
      position: relative;
    }

    .text {
      position: absolute;
      top: 30px;
      left: 530px;
      font-family: initial;
      text-align: center;
    }
