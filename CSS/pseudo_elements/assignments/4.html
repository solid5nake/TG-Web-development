Opdracht 4

Hieronder vind je een voorbeeld van het gebruik van een pseudo-element. Kun jij dit namaken?

Hiervoor zijn 2 div elementen aangemaakt naast de button element, de button heeft de pseudo-elementen ::before (voor de button) en ::after (achter de button) meegekregen
Een "div" wordt als waarde ingesteld voor de content property.

Note: De ::before en ::after pseudo-elementen hebben de property content nodig om iets weer te geven. 

    CSS
    button {
      position: relative;
      border: initial;
      background: rgb(85, 175, 170);
      width: 910px;
      height: 55px;
      font-size: 30px;
      color: white;
      margin: 0;
    }

    button::before {
      width: 50px;
      height: 50px;
      position: absolute;
      display: block;
      top: 0;
      left: 0;
      content:"div" ;
      color:rgb(85, 175, 170);
      border-top: 5px solid orange;
      border-left: 5px solid orange;
    }

    button::after {
      width: 50px;
      height: 50px;
      position: absolute;
      display: block;
      bottom: 0;
      right: 0;
      content:"div" ;
      color:rgb(85, 175, 170);
      border-bottom: 5px solid orange;
      border-right: 5px solid orange;
    }
