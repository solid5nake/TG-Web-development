		CSS selectors worden gebruikt om HTML elementen te selecteren die je wilt stylen.
		We kunnen de CSS selectors in vijf categorien verdelen.

CSS selector categorien

  • Simple selectors (selecteerd elementen op basis van naam, id, class)
  • Combinator selectors (selecteerd elementen op basis van een specifieke relatie onderling)
  • Pseudo-class selectors (selecteerd elementen op basis van een bepaalde state)
  • Pseudo-elements selectors (selecteerd en styled een deel van een element)
  • Attribute selectors (selecteerd elementen op basis van een attribute of de waarde van een attribute)
