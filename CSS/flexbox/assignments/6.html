Opdracht 6

    Maak het onderstaande na met flexbox. Properties die je o.a dient te gebruiken zijn:
  • display
  • justify-content
  • flex-direction

Code

    .container > div {
      flex-basis: 3.6em;
      background: rgb(170, 176, 248);
    }

    .container {
      display: flex;
      flex-direction: column;
      justify-content: space-between;
      padding: 6.250em 0 6.250em 0;
      width: 114em;
      height: 25em;
      background: rgb(254, 250, 210);
    }

    .row1{
      width: auto;
    }
    .row2{
      width: 49.13em;
    }

      width: 60.63em;
