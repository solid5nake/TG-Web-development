<h1> element krijgt een position:relative waarmee je het relatief positioneerd
daarbij hoor je het in spiegelbeeld (top en right) te schrijven om het linksonder te positioneren. 

    CSS
    h1 ~ .example {
      position: relative;
      right: 20px;
      top: 30px;
    }
