Alt attribute in de HTML image tag

De alt attribute geeft de image een beschrijving, het is niet verplicht om te gebruiken maar wel heel handig voor de toegangklijkheid Screenreaders lezen deze beschrijving voor aan de (blinde) lezer, zodat deze weet om wat voor afbeelding het gaat De alt text wordt ook weergegeven op het scherm als de image niet geladen kan worden.
