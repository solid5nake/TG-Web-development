    Wat gebeurt er als je een width en een height toevoegt aan de span tag (  ) van vorige opdracht?
  
Er gebeurd niets met de span.

Dit is een block box en neemt de hele breedte in waardoor de volgende box opde volgende regel zit

In dit blok zit inline tekst in wat dus niet op de volgende regel komt te staan

De span tag is een inline block, om die reden zijn de width en height niet toepasbaar
