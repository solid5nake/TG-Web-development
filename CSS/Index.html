CSS

Inleiding

Cascading Style Sheet

      CSS is een stylesheet-language die wordt gebruikt om de presentatie van een document te beschrijven dat is geschreven in HTML of XML (inclusief XML-dialecten zoals SVG, MathML of XHTML). 

      CSS beschrijft hoe elementen op het scherm, op papier, in spraak of op andere media moeten worden weergegeven.

Leerdoelen

  • Selectors & visual rules leren over het opmaken van individuele elementen en groepselementen met behulp van verschillende visuele CSS-regels.
  • Boxmodel leren hoe het box-model gebruikt wordt om HTML-elementen op webpagina’s te plaatsen.
  • Display & positioning leren over de CSS-regels voor het weergeven en positioneren van elementen op webpagina’s.
  • Pseudo elements leren over de werking en het gebruiken van pseudo elementen als krachtige tools om je webpagina extra stijl te geven.
  • Flexbox Leren over het gebruik van flexbox en de voordelen die het heeft ten opzichte van het gewone box model.
  • Transitions / Animations Leren over hoe je met transitions en animations het gedrag en uiterlijk van elementen kunt beïnvloeden zonder gebruik te hoeven maken van Javascript.
  • Media queries Leren over hoe je een pagina responsive kan maken, zodat het op meerdere scherm goed uitziet.
  • Eindopdracht De eindopdracht is het maken van een volledige webpage. Het is de bedoeling dat men kiest uit een aantal templates en de gekozen template gaat nabouwen.
