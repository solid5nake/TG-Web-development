Pseudo Class Selectors

Pseudo class selectors zijn CSS selectors die beginnen met een colon(:) teken
Een pseudo-class selector targets elements afhankelijk van hun state ipv de informatie vanuit de DOC tree.
Neem bijvoorbeeld :hover deze wordt alleen getriggered wanneer de muis over het element heengaat.

css a:hover {

}

Syntax

css selector:pseudo-class {
  property: value;
}
