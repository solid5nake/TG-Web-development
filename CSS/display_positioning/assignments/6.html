Display & Positioning

Opdracht 6

    Maak het onderstaande na door gebruik te maken van relative position. De outer div block is
    100px breed en 100px hoog. De inner div block is 50px breed en 50px hoog.
  
    .outer {
      position: relative;
      background-color: orange;
      width: 100px;
      height: 100px;
    }
    .inner {
      background-color: springgreen;
      width: 50px;
      height: 50px;
      position: absolute;
      top: 25px;
      left: 25px;
    }
  
    HTML
    < class="outer">
      < class="inner">
    </div>
  
    Hierbij heeft de outer div block position relative en de inner div block position absolute. 

    De inner div block zal zich relatief positioneren van zijn containing block.
