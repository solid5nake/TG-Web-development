Opdracht 8

    Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height
    de totale breedte en hoogte is van het blok van de voorgaande opdracht?

box-sizing property

    De box-sizing CSS property stelt in hoe de totale width en height van een element wordt ingecalculeerd. 

    Per default in een CSS boxmodel zijn de width en height alleen van toepassing bij de content van het element. 

    Als er ook nog eens padding en margin zijn toegevoegd dan zal dit ook meeberekend worden.
