slice()

  
    slice() method geeft een oppervlakkige kopie van een deel van een array terug naar een nieuw array-object dat van begin tot eind is geselecteerd (einde niet inbegrepen), waarbij begin en einde de index van items in die array vertegenwoordigen. De originele array wordt niet gewijzigd.
  

  
arr.slice([start[, end]])

  
parameters

  
start (optioneel)

  
    Zero-based index om de extractie te starten.
    Er kan een negatieve index worden gebruikt, die een afwijking aangeeft vanaf het einde van de reeks. slice (-2)extraheert de laatste twee elementen in de reeks.
    Als start niet is gedefinieerd, begint slice vanaf de index 0.
    Als start groter is dan het indexbereik van de reeks, wordt een lege array geretourneerd.
  

  
end (optioneel)

  
    Zero-based index waarvoor de extractie moet worden beëindigd. slice() extraheert tot en met einde, het einde wordt niet geincludeerd. Plak (1,4) extraheert bijvoorbeeld het tweede element tot en met het vierde element (elementen met de index 1, 2 en 3).
    Er kan een negatieve index worden gebruikt, die een afwijking aangeeft vanaf het einde van de reeks. slice (2, -1) extraheert het derde element via het op een na laatste element in de reeks.
    Als einde wordt weggelaten, plak extracten door het einde van de reeks (arr.lengte).
    Als het einde groter is dan de lengte van de reeks, snijdt u extracten door tot het einde van de reeks (arr.lengte).

let arrDeletedItems = array.splice(start[, deleteCount[, item1[, item2[, ...]]]])

    console.log(fruits);
    console.log(fruits.splice(0, 3, 'rambutan'));
    console.log(fruits);

    (5) ["kiwi", "peer", "aardbei", "mandarijn", "mango"]
    (3) ["kiwi", "peer", "aardbei"]
    (3) ["rambutan", "mandarijn", "mango"]
  

  
de start index geeft aan waar de vervangende element komt, het 2e getal geeft aan het aantal elementen dat verwijderd moet worden van start.

    sort() method sorteerd de elementen van een array in place en retourneerd de gesorteerde array.
    De standaard sorteervolgorde is oplopend, gebaseerd op het omzetten van de elementen in strings en het vergelijken van hun UTF-16 code unit values.
  

  
arr.sort([compareFunction])

  
Code

    const fruits = ["appel", "peer", "aardbei", "mandarijn", "mango"];

    5) ["aardbei", "appel", "mandarijn", "mango", "peer"]
  

  
de array wordt op alfabetische volgorde gesorteerd, echter kan sort() het ook numeriek sorteren.

    Opdracht 9:

    Kijkend naar de fruits array. Hoe draai je de values “Aardbei” en “Mandarijn” om in de array door gebruik te maken van indices (index). 
  

  
    De onderstaande functie switched twee array items die naast elkaar zijn, de eerste item wordt gevonden aan de hand van zijn index en wordt bewaard in een variable, vervolgens wordt met de splice() dezelfde index
    herbruikt als start punt, vanaf die startpunt wordt er 1 item verwijderd, bij de volgende regel wordt met de splice() method de variable op index + 1 geplaatst.
  

  
Code

  
    const fruits = ["appel", "peer", "aardbei", "mandarijn", "mango"];

    function switchFruit(arr, i) {
      const fruit = arr[i];
      fruits.splice(i, 1);
      fruits.splice((i + 1), 0, fruit);
      return arr;
    }

    switchFruit(fruits, 2);
    5) ["appel", "peer", "mandarijn", "aardbei", "mango"]
  

  
i staat voor index
