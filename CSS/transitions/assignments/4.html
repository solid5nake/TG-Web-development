Transitions

Opdracht 4

    Je kunt op verschillende properties transitions aangeven. Zie hieronder een voorbeeld. Zoek uit op welke properties
    de transitions zitten en maak het ongeveer hetzelfde na.
1
2
3
    Voor elke element en property heb je een andere transitions, sommige hebben zelfs een
    transition-delay.

Code

    .transitions-container {
      position: relative;
      height: 500px;
      width: 1000px;
      border: 3px solid black;
      font-family: 'Times New Roman', Times, serif;
      font-size: 39px;
    }

    .transitions-container>div {
      display: flex;
      position: absolute;
      justify-content: center;
      align-items: center;
      top: calc((50% - 0.75em) - 6px);
      left: calc((50% - 1.25em) - 6px);
      width: 1.5em;
      block-size: 1.5em;
      border: 3px solid black;
    }

    .transitions-container:hover .box-1,
    .transitions-container:hover .box-2 {
      top: 0;
      transition: top 1s cubic-bezier(.17, .67, .62, .15), left 650ms cubic-bezier(.17, .67, .21, .94) 350ms;
    }

    .transitions-container:hover .box-1 {
      left: calc((100% - 1.5em) - 6px);
    }

    .transitions-container:hover .box-2,
    .transitions-container:hover .box-3 {
      left: 0;
    }

    .transitions-container:hover .box-3 {
      top: calc((100% - 1.5em) - 6px);
      transition: left 0.5s linear 200ms, top 0.5s linear 0.7s;
    }

    .box-1,
    .box-2 {
      transition: top 1s cubic-bezier(.17, .67, .62, .15), left 650ms cubic-bezier(.17, .67, .21, .94) 350ms;
    }

    .box-3 {
      transition: left 0.5s linear 200ms, top 0.5s linear 0.7s;
    }

transition: transition-property, transistion-duration, transition-timing-function, transition-delay
