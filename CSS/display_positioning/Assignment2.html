Opdracht 2

Zie de onderstaande HTML code.
Hoe zorg je er voor dat het <h1> element altijd 50px van de top en 50px van de rechterkant van het scherm zichtbaar is. Maak het <h1> element rood door een RGB kleurcodering toe te passen. 

    HTML
    <h1>Dit is een titel</h1>
    <p>Dit is een paragraaf.</p>
    <p>Dit is nog een paragraaf.</p>

Dit is een titel

Dit is een paragraaf.

Dit is een paragraaf.

Dit is nog een paragraaf.

Door een position property fixed te declareren wordt de h1 rechtboven in de viewport positioneerd. 

    HTML
    <h1 class="example">Dit is een titel</h1>
    <p>Dit is een paragraaf.</p>
    <p>Dit is nog een paragraaf.</p>
  
    CSS
    h1 ~ .example {
      position: fixed;
      top: 50px;
      right: 50px;
      color: rgb(255, 0, 0);
    }
