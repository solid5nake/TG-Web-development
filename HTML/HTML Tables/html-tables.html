HTML tables

  • table tag, definieert een html tabel
  • thead tag, wordt gebruikt om een headers content te groeperen van een tabel
  • tbody tag, wordt gebruikt om de body content te groeperen in een html tabel
  • tfoot tag, groepeert footer content in een HTML tabel
  • th tag, definieert een header cel in een HTML tabel
  • tr tag, definieert een regel in een HTML tabel
  • td tag, definieert een standaard data cel in HTML tabel
