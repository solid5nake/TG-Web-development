Flexbox

Opdracht 6

Maak het onderstaande na met flexbox. Properties die je o.a dient te gebruiken zijn:

  • display
  • justify-content
  • flex-direction

Code

    CSS
    .container > div {
      flex-basis: 3.6em;
      background: rgb(170, 176, 248);
    }

    .container {
      display: flex;
      flex-direction: column;
      justify-content: space-between;
      padding: 6.250em 0 6.250em 0;
      width: 114em;
      height: 25em;
      background: rgb(254, 250, 210);
    }

    .row1{
      width: auto;
    }
    .row2{
      width: 49.13em;
    }
    .row3{
      width: 60.63em;
    }
    .row4{
      width: 72em;
    }
    .row5{
      width: 26.38em;
    }
    .row6{
      width: 37.81em;
    }

div.container heeft de properties (display:flex; flex-direction:column; justify-content:space-between;)
div heeft een flex-basis declaratie.
