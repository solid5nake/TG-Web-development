Opdracht 1

Code

    HTML
    <div class="container">
      <div class="box">1</div>
      <div class="box">2</div>
      <div class="box">3</div>
      <div class="box">4</div>
      <div class="box">5</div>
      <div class="box">6</div>
    </div>
  
    CSS
    .container {
      display: flex;
      width: 900px;
      margin: 50px;
      background: rgb(251, 254, 183);
    }
    
    .box {
      display: flex;
      margin: 15px;
      flex-basis: 90px;
      block-size: 90px;
      justify-content: center;
      align-items: center;
      background: rgb(201, 251, 182);
      font-family: 'Times New Roman', Times, serif;
      font-size: 25px;
    }

De parent heeft een display:flex property om het een flexbox van te maken. De child heeft een display:flex property om de tekst in het midden te centreren zonder gebruik te hoeven maken van line-height, de properties justify-content (horizontale as) en align-items (verticale as) worden ingesteld op center. De flex-basis property set de initial main size van een flex item (width).

Note: de block-size property is toegepast om de height in te stellen.
