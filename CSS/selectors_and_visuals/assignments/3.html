klik op 1 van de bovenstaande links

Type selector

    p { /* "p" is the type selector */
        margin: 0 0 1em 0;
    }
  
		Een Type Selector (ook wel gerefereerd als een Element Type Selector) matched elements met de
		corresponderende element node name, zoals { p } , { span } ,{ div } tags.
		Type selectors worden in het algemeen gebruikt om in 1 keer veranderingen aan te brengen aan de style van een website

Descendant selector

		ul li { } 
		header h2 { }
		footer a { }
		.modue div { }
		#info-toggle span { }
		div dl dt a { }
	
		Een Descendant Selector in CSS is een selector met een spatie tussen twee selectors zonder een
		combinator.

Class selector

		.class { }
	
		Een class selector selecteerd alle elementen met een matchende class attribuut.
