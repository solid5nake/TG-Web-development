Opdracht 6

    Hieronder vind je wederom een voorbeeld van het gebruik van een pseudo-element. Kun jij dit namaken?
1
2
3

Code

    <div class="container">
      <div class="sphere">1</div>
      <div class="sphere">2</div>
      <div class="sphere">3</div>
    </div>
  
    .sphere::before {
      position: absolute;
      border-radius: 50%;
      top: 50px;
      left: -35px;
      line-height: 60px;
      content: "✪";
      background: black;
      width: 80px;
      height: 80px;
      font-size: 15px;
    }

    .sphere::after {
      position: absolute;
      border-radius: 50%;
      top: 50px;
      right: -35px;
      content: "❖";
      line-height: 100px;
      background: black;
      width: 80px;
      height: 80px;
      font-size: 15px;
    }

Resultaat

1
2
3

De pseudo elementen zijn absolutely positioned relatively from it's nearest ancestor (.container).
De content property values van de pseudo-elementen (.sphere::before, .sphere::after) zijn ingesteld op speciale karakters "✪", "❖" .

Note: Om de speciale karakters in de pseudo-elementen op hoogte te positioneren is de CSS property line-height gebruikt.

line-height

    De CSS-eigenschap line-height stelt de hoogte van een line box in. Het wordt vaak gebruikt om de afstand tussen
    regels tekst in te stellen. Op bloklevel-elementen specificeert het de minimale hoogte van line boxes binnen het
    element. Op non-replaced inline-elementen specificeert het de hoogte die wordt gebruikt om de hoogte van het line
    box te berekenen.
