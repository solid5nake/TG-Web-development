    Als je ruimte wilt maken tussen de border en content, welke css property gebruik je dan?
    Maak een voorbeeld in je antwoord.

box

    Hiervoor wordt de property padding gebruikt, de div element heeft de property padding meegekregen
    met een waarde van 10px, de background illustreert heel mooi de ruimte die is gecreeerd met padding.

Code

      <div class="example">
        <h3>box</h3>
      </div>
    
      .example {
        text-align: center;
        border: 0.3rem solid black;
        width: fit-content;
        background: moccasin;
        padding: 0.5rem;
      }
  
      .example h3 {
        border: 0.3rem solid black;
        background: #FFF;
        color: black;
      }
