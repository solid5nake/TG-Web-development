Opdracht 7

    Maak het onderstaande na met flexbox. Properties die je o.a dient te gebruiken zijn:
  • display
  • justify-content
  • flex-direction
  • align-items

Code

    <div class="container">
      <div class="column1"></div>
      <div class="column2"></div>
      <div class="column3"></div>
      <div class="column4"></div>
      <div class="column5"></div>
      <div class="column6"></div>
    </div<
  
    .container {
      display: flex;
      flex-basis: initial;
      flex-direction: row;
      padding: 0.35em;
      align-items: center;
      justify-content: flex-start;
      height: 60em;
      width: 47em;
      background: rgb(254, 250, 210);
    }

    .column1{
      height: 100%;
    }
    .column2{
      height: 28.125em;
    }
    .column3{
      height: 33.875em;
    }
    .column4{
      height: 39.688em;
    }
    .column5{
      height: 16.438em;
    }
    .column6{
      height: 22.188em;
    }
  
    div.container properties (flex-direction: row;, align-items: center;, justify-content: flex-start; ) 

    div.column properties hebben ieder een eigen height meegekregen.
