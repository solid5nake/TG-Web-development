::first-letter

Een CSS-pseudo-element die stijlen toe past op de eerste letter van de eerste regel van een block-level element, maar alleen als deze niet wordt voorafgegaan door andere inhoud (zoals afbeeldingen of inline-tabellen).

Note: Een user-agent is een computerprogramma dat een persoon vertegenwoordigt, bijvoorbeeld een browser in een webcontext.

::grammer-error

Een CSS-pseudo-element representeerd een tekst segment dat een user agent ziet als een grammatische fout.
Slechts een kleine subset van CSS-eigenschappen kan worden gebruikt in een regel met :: grammar-error in zijn selector:

  • color
  • background-color
  • cursor
  • caret-color
  • outline
  • text-decoration
  • text-emphasis-color
  • text-shadow
HTML <p>My friends is coming to the party tonight.</p> 
    CSS
    ::grammar-error  {
      text-decoration: underline red;
      color: red;
    }

::marker

De ::marker CSS pseudo-element selecteerd de marker box van een list item dat
bullets nummers of bevat. Het werkt op elke element of pseudo-element dat gesteld is op display:list-item
zoals de <li> en <summary> elementen. 

    CSS
    ::marker {
      color: blue;
      font-size: 1.2em;
    }

Toegestaande properties

  • Alle font properties
  • white space properties
  • color
  • text-combine-upright, unicode-bidi en direction
  • content property
  • Alle animation en transition properties

::part()

Het ::part() CSS pseudo-element vertegenwoordigt elk element binnen een shadowtree dat een overeenkomend part attribute heeft.

Een shadow tree is een boom van DOM-nodes waarvan de boenste node een shadow root is;
dat wil zeggen, het bovenste node binnen een shadow-DOM. Een shadowtree is een verborgen set standaard DOM-knooppunten die is gekoppeld aan een standaard DOM-node dat als host dient.

Syntax

    ::part( <ident>+ )
  
    HTML
    <template id="tabbed-custom-element">
      <style type="text/css">
      *, ::before, ::after {
        box-sizing: border-box;
        padding: 1rem;
      }
      :host {
        display: flex;
      }
      </style>
      <div part="tab active"Tab 1</div>
      <div part="tab">Tab 2</div>
      <div part="tab">Tab 3</div>
      </template>
  
    CSS
    tabbed-custom-element::part(tab) {
      color: #0c0dcc;
      border-bottom: transparent solid 2px;
    }
    
    tabbed-custom-element::part(tab):hover {
      background-color: #0c0d19;
      border-color: #0c0d33;
    }
    
    tabbed-custom-element::part(tab):hover:active {
      background-color: #0c0d33;
    }
    
    tabbed-custom-element::part(tab):focus {
      box-shadow: 
        0 0 0 1px #0a84ff inset, 
        0 0 0 1px #0a84ff,
        0 0 0 4px rgba(10, 132, 255, 0.3);
    }
    
    tabbed-custom-element::part(active) {
      color: #0060df;
      border-color: #0a84ff !important;
    }
  
    JAVASCRIPT
    let template = document.querySelector("#tabbed-custom-element");
    globalThis.customElements.define(template.id, class extends HTMLElement {
      constructor() {
        super();
        this.attachShadow({ mode: "open" });
        this.shadowRoot.appendChild(template.content);
      }
    });

::placeholder

De ::placeholder CSS-pseudo-element vertegenwoordigt de tijdelijke aanduiding voor tekst (placeholder) in een <input> - of <textarea> element. 

    CSS
    ::placeholder {
      color: blue;
      font-size: 1.5em;
    }

Ze zijn handig om te gebruiken bij een user input, placeholders in formfields is niet aan te raden. 

    <label for="user-email">Your email address</label>
    <span id="user-email-hint" class="input-hint">Example: jane@sample.com</span>
    <input id="user-email" aria-describedby="user-email-hint" name="email" type="email">

::selection

past stijlen toe op het deel van een document dat door de gebruiker is gemarkeerd (zoals klikken en slepen met de muis over tekst). 

    CSS
    ::selection {
      background-color: green;
    }

Toegestaande properties

Alleen bepaalde CSS-eigenschappen kunnen worden gebruikt met :: selection:

  • color
  • background-color
  • cursor
  • caret-color
  • outline
  • text-decoration
  • text-emphasis-color
  • text-shadow

::slotted()

Het ::slotted() CSS pseudo-element vertegenwoordigd elke element dat in een slot is geplaatst binnen in en HTML template.

Dit werkt alleen bij gebruik in CSS geplaatst in een schaduw-DOM. 

    /* Selects any element placed inside a slot */
    ::slotted(*) {
      font-weight: bold;
    }

    /* Selects any <span> placed inside a slot */
    ::slotted(span) {
      font-weight: bold;
    }
*Selecteert elk element dat in een sleuf is geplaatst *Selecteert een <span> geplaatst in een slot

::spelling-error

Het :: spelling-error CSS pseudo-element vertegenwoordigt een tekstsegment dat door de user agent is gemarkeerd als onjuist gespeld. 

    CSS
    ::spelling-error  {
      text-decoration: wavy red;
    }

Toegestaande properties

Alleen bepaalde CSS-eigenschappen kunnen worden gebruikt met :: selection:

  • color
  • background-color
  • cursor
  • caret-color
  • outline
  • text-decoration
  • text-emphasis-color
  • text-shadow

::cue-region

Het ::cue-region CSS pseudo-element komt overeen met WebVTT-cues binnen een geselecteerd element. Dit kan worden gebruikt om bijschriften en andere aanwijzingen in media te stylen met VTT-tracks.

CSS ::cue-region { color: yellow; font-weight: bold; }

Syntax

    ::cue-region | ::cue-region(  )

De eigenschappen worden toegepast op de hele set cues alsof ze een enkele eenheid zijn.
De enige uitzondering is dat de achtergrond en de shorthand properties ervan op elke cue afzonderlijk van toepassing zijn, om te voorkomen dat er boxen worden gemaakt en onverwacht grote delen van de media worden verduisterd.
